Règles du jeu

Le but du jeu est simple, réussir à reproduire tous les emojis à l'intérieur du carré de couleur en prenant le moins de temps possible

Basé sur l'intelligence artificielle

Le but du projet est de pouvoir implémenté un modèle de détection du côté client avec TensorFlow, c'est à dire d'avoir un modèle de Machine Learning qui utilise les ressources du client plutôt que les ressources d'un server
